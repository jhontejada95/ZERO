## Description

ZERO is a prevention-finance protocol that turns verified prevention into a fundable, auditable and payable outcome.

It connects evidence, counterfactual analysis, independent verification, human authorization and blockchain settlement through a new financial primitive: the Prevention Receipt.

Agent proposes.  
Human authorizes.  
Contract settles.  
Auditor reconstructs.

The functional MVP includes:

- Role-scoped stakeholder experiences
- AWS DynamoDB event persistence
- Coinbase Developer Platform wallets
- x402 evidence payments
- EIP-712 human authorization
- Ethereum Attestation Service
- Smart contracts and confirmed transactions on Base Sepolia
- Vercel deployment and serverless APIs

This project and video were created for the purpose of entering the H0 Hackathon.
